# 📊 Customer Churn Analysis (Power BI)

## 📌 About This Project
Hi! 

This project focuses on analyzing customer churn data using Power BI to understand the reasons behind customer departures from a business.

The goal was to explore customer behavior, identify patterns, and provide insights that can help improve customer retention.

---

## 🎯 Objectives
- Understand the overall churn rate  
- Identify factors that influence customer churn  
- Analyze customer behavior and service usage  
- Provide recommendations to reduce churn  

---

## 🛠️ Tools Used
- Power BI  
- Excel / CSV dataset  
- Data cleaning and analysis  

---

## 📊 Dashboard Overview
The dashboard includes:

- Total customers, churned customers, and churn rate  
- Customer segments (gender, partners, dependents)  
- Service usage (internet type, security, backup, streaming)  
- Contract types  
- Payment methods  

---

## 🔍 Key Insights
Some of the main findings:

- The churn rate is **27%**, which is relatively high  
- Customers on **month-to-month contracts** are more likely to leave  
- **Fiber optic users** have higher churn compared to other service types  
- Customers without **online security and additional services** are more likely to churn  
- Customers without partners or dependents tend to leave more  

---

## 💡 Recommendations
Based on the analysis, the business can:

- Encourage customers to move to long-term contracts  
- Improve service quality, especially for fiber users  
- Offer bundled services (security, backup, etc.)  
- Target high-risk customers with retention strategies  
- Review pricing and customer experience  

---

## ▶️ How to Use This Project
1. Download the Power BI file (.pbix)  
2. Open it in Power BI Desktop  
3. Use filters and visuals to explore the data
